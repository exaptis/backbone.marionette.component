Backbone.Marionette.Component

Based on Apache Wicket and Marionette.js

Backbone.Marionette.Component is an extension library for Marionette.js which lets you use web components for handling forms.

If you like this project you can support me by starring this repository, or reporting bugs and ideas.

Read more about Marionette.Component for Backbone at the Github page.
